# filehide
Esconda arquivos em imagens!

Necessário o Perl, RAR e UNRAR.

Para ajuda

./filehide -h

http://i.imgur.com/KWZzf9u.png

Criar arquivo:

./filehide -m compactado.rar foto.jpg imageninofensiva.jpg

Testar:

./filehide -t mageninofensiva.jpg

Extrair:

./filehide -x mageninofensiva.jpg
